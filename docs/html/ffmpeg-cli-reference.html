FFMPEG CLI

VERSION 8.0.1 | ARM64 | APPLE SILICON

FFmpeg is the leading multimedia framework, able to decode, encode, transcode, mux, demux, stream, filter and play pretty much anything that humans and machines have created.

01INSTALLATION

02BASIC SYNTAX

ffmpeg [global_options] {[input_options] -i input_file}... {[output_options] output_file}...

Common Patterns

ffmpeg -i input.mp4 output.mkv                        # BASIC CONVERSION
ffmpeg -i input1.mp4 -i input2.mp3 output.mp4         # MULTIPLE INPUTS
ffmpeg -y -i input.mp4 output.mp4                     # OVERWRITE OUTPUT
ffmpeg -n -i input.mp4 output.mp4                     # NO OVERWRITE

04VIDEO OPTIONS

OptionDescription
-r[:stream_spec] <rate>Set frame rate (Hz value, fraction or abbreviation)
-aspect[:stream_spec] <ratio>Set aspect ratio (4:3, 16:9 or 1.3333, 1.7777)
-vnDisable video
-vcodec <codec>Set video codec (alias for -c:v)
-vf <filter_graph>Set video filter chain (alias for -filter:v)
-b <bitrate>Set video bitrate (use -b:v, e.g., -b:v 5M)
-pix_fmt[:stream_spec] <format>Set pixel format (yuv420p, nv12, rgb24, etc.)
-vframes <number>Set the number of video frames to output
-crf <float>Constant quality factor (x264/x265, 0-51, lower is better)
-preset <name>Encoding preset (ultrafast to veryslow)

05AUDIO OPTIONS

OptionDescription
-aq <quality>Set audio quality (codec-specific)
-ar[:stream_spec] <rate>Set audio sampling rate (Hz), common: 44100, 48000
-ac[:stream_spec] <channels>Set number of audio channels (1=mono, 2=stereo, 6=5.1)
-anDisable audio
-acodec <codec>Set audio codec (alias for -c:a)
-ab <bitrate>Set audio bitrate (use -b:a, e.g., -b:a 128k)
-af <filter_graph>Set audio filter chain (alias for -filter:a)

08HARDWARE ACCELERATION

Native Support: VideoToolbox (Apple Silicon / macOS)

# Use VideoToolbox hardware decode
-hwaccel videotoolbox -i input.mp4

# Use VideoToolbox hardware encode H.264
-c:v h264_videotoolbox -b:v 5M

# Use VideoToolbox hardware encode H.265
-c:v hevc_videotoolbox -b:v 10M -q:v 50

# Full hardware acceleration pipeline
-hwaccel videotoolbox -hwaccel_output_format videotoolbox_vld -i input.mp4 -c:v hevc_videotoolbox output.mp4

09EXAMPLES

Format Conversion

# MP4 TO MKV
ffmpeg -i input.mp4 -c copy output.mkv

# MKV TO MP4 (RE-ENCODE)
ffmpeg -i input.mkv -c:v libx264 -c:a aac output.mp4

# VIDEO TO GIF
ffmpeg -i input.mp4 -vf "fps=15,scale=480:-1:flags=lanczos,split[s0][s1];[s0]palettegen[p];[s1][p]paletteuse" output.gif

# VIDEO TO WEBM
ffmpeg -i input.mp4 -c:v libvpx-vp9 -crf 30 -b:v 0 -c:a libopus output.webm

Video Processing

# COMPRESS VIDEO (CRF MODE)
ffmpeg -i input.mp4 -c:v libx264 -crf 23 -preset medium -c:a copy output.mp4

# SCALE RESOLUTION
ffmpeg -i input.mp4 -vf "scale=1280:720" -c:a copy output.mp4

# TRIM VIDEO
ffmpeg -i input.mp4 -ss 00:01:30 -t 30 -c copy output.mp4

# ROTATE 90 DEGREES
ffmpeg -i input.mp4 -vf "transpose=1" -c:a copy output.mp4

# ADD WATERMARK
ffmpeg -i input.mp4 -i watermark.png -filter_complex "overlay=W-w-10:H-h-10" -c:a copy output.mp4

Audio Processing

# EXTRACT AUDIO
ffmpeg -i input.mp4 -vn -c:a libmp3lame -q:a 2 output.mp3

# ADJUST VOLUME
ffmpeg -i input.mp3 -af "volume=10dB" output.mp3

# AUDIO FADE IN/OUT
ffmpeg -i input.mp3 -af "afade=in:st=0:d=5,afade=out:st=55:d=5" output.mp3

# MERGE MULTIPLE AUDIO
ffmpeg -i audio1.mp3 -i audio2.mp3 -filter_complex "[0:a][1:a]concat=n=2:v=0:a=1" output.mp3
